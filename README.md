# What is Nolann's Player Health Indicator?

Nolann's Player Health Indicator is a fork of Gaider10's mod that add a config menu and support for the newest versions of the game !

##  What's new in this version of the mod?

This version takes an idea from [HuTAR](https://modrinth.com/user/HuTAR) to allow you to see the hearts of invisible players only when they are wearing a piece of armor. The mod also adds the option to see your own hearts in third-person view via the new config screen.

_The config screen is only available if you have the [Cloth Config API](https://modrinth.com/mod/cloth-config) mod installed!_
